DigitalOcean Serverless Tools Download Space

This DigitalOcean space is a place from which serverless tools can be installed. It is currently updated manually when tools change. This will be automated eventually. The site may be moved in the future.

To download the most recent build of the nim CLI:

  • Installer for MacOs.
    • The installer is unsigned and may give security warnings.
  • Script for network install on MacOs.
    • You might find this more convenient than using the installer.
  • Installer for Windows.
    • The installer is unsigned and may give security warnings, both in the browser and when you open the installer.
    • There is currently no script alternative for Windows.
  • Script for network install on Linux.
    • There is no installer for Linux equivalent to those for MacOS and Windows.

"Latest" builds are not explicitly versioned (they keep the version of the last stable release).
